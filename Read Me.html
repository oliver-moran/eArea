eArea - a simple web-based text editor
Copyright © 2006, 2008 Oliver Moran

eArea is a really simple web-based WYSIWYG text editor. It works with almost all popular browsers, including Internet Explorer, Firefox/Mozila/Camino/Netscape, Konqueror/Safari and Opera. It even with OS X Dashboard widgets! With a short snippet of HTML you can insert eArea onto any webpage and fully customise it for your site. You can even have as many eAreas as you like on a page and customise them all independently. If eArea comes across a browser it doesn't support (like older versions of Opera), it will display an ordinary <textarea> instead, but its appearance will still be as you defined.

How To Use eArea

Putting eArea onto a webpage is really simple:

  1. Paste the HTML snippet into your webpage where you want an eArea to appear
  2. Edit the snippet code to define the size of the eArea, which toolbar buttons should appear, etc.
  3. Copy the eArea folder into the same directory as your webpage

Everything about eArea is customisable. The settings for each eArea are defined in its snippet:

  • An individual name for each eArea
  • The eAreas width and height
  • The layout of the eArea: what buttons should appear in its toolbar, how they should be laid-out, etc.
  • The default font style for your eArea

A helper function is included with each eArea. If you call editAreaValue("foo") from JavaScript, it will return the HTML content of an eArea called foo.
